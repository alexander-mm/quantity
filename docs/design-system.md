## Typography Rules

Only one Page Title per page.

Do not use more than two font weights in the same component.

Body text must use 14px by default.

Small text is only for secondary information.

Use Bold only for page titles.

## Spacing Rules

Never use arbitrary spacing values.

Always use the defined spacing scale.

Maintain consistent spacing between similar components.

Related elements should stay visually grouped.

Large sections require more whitespace than small controls.
